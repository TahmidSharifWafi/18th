Task-1:

Here is an array, a=[1,2,34,67,3,21,1 ,4,6].
You have to find the sumation of this array's even numbers'.

Task-2:

Here is an array, b=[3,4,8,2,1,98,6,5 ,7,6].
You have to find the sumation of this array's odd numbers'.



Task-1 Answer:

The sumation of the even numbers of the array named a is: 46

a = [1, 2, 34, 67, 3, 21, 1, 4, 6]
sum = 0
for (var i = 0; i < a.length; i++) {
if (a[i]%2==0) {
sum = sum + a[i]
}
}
console.log(sum)

You can also see the ans in console.
